# 九九乘法表射击游戏 (Multiplication Table Shooting Game)

这是一个适合小朋友学习九九乘法表的网页射击游戏。

## 游戏玩法

- **目标**：击落飘落的角色。
- **操作**：
  - 观察角色上的乘法题（如 `6 x 7`）。
  - 点击或触摸屏幕底部对应的 **正确答案** 选项。
  - 答对发射子弹击落角色。
- **结束**：
  - 如果角色落地，游戏结束。
  - 如果选择错误答案，游戏结束。

## 游戏特色

- **角色选择**：开始游戏前可以选择喜欢的角色（🎈 气球、🐰 兔子、🐻 小熊）。
- **难度选择**：提供慢速、普通、快速三种难度。

## 技术栈

- 纯原生 HTML5 + CSS3 + JavaScript
- Canvas API 渲染
- 无任何第三方依赖，单文件 `index.html` 即可运行

## 如何运行

1. 下载本项目。
2. 直接用浏览器（Chrome/Safari/Edge/Firefox）打开 `index.html`。
3. 支持 PC 鼠标和手机触摸屏。

## 开发与修改

所有代码均位于 `index.html` 中，注释清晰，易于修改参数（如角色速度、生成间隔等）。
